# Mobile Screen 1 — Contact Ombuds

Purpose:
Allow a visitor to send a single confidential message with minimal friction.

UI Elements:
- Institution name (auto-loaded via QR or dropdown)
- Role selection buttons:
  - Student
  - Staff
  - Faculty
- Large multiline text box:
  Placeholder: "Tell the Ombuds what’s going on."
- Submit button:
  Label: "Send confidential message"

Rules:
- No validation beyond non-empty message
- No categories
- No severity flags
- No required personal info
