# Digital Ombuds Mobile App (Visitor)

This folder will contain the Expo-based mobile application for visitors.

## MVP Scope
- Institution selection (QR or dropdown)
- Role selection (student / staff / faculty)
- Single free-text message submission
- Secure encrypted message thread
- Recovery key support
- Push notifications (optional for MVP)

## Tech Stack (Planned)
- Expo (React Native)
- Supabase client (auth + messaging)
- Secure storage for session tokens

## Build Philosophy
- Cloud-first development
- Minimal storage footprint
- No visitor accounts
- Anonymous-by-design
- Extremely simple UX

Not yet implemented.
